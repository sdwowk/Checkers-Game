Checkers Rules and Need to knows:
	1. To run the game:
		-To play against the AI that does a random move each
		turn write ipython3 main.py CheckersBoard into the terminal whilst inside the p2_1276067 directory.
		-To play against the AI the "Smart" AI write
		ipython3 mainSmart.py CheckersBoard into the terminal
		whilst in the p2_1276067 directory.
	2. Starting the game:
		To start the game the player must choose heads or tails
		from the buttons on the bottom right of the game. The user is the team with GREY pieces. If the red pieces
		do not move that means the user won the "coin toss"
		and must move first.
	3. Moving pieces:
		To move pieces the user selects the pieces using the
		mouse.

	4. Jumping units:
		If a jump is available to one of the pieces, the user must move the piece that can jump.
		To perform a double jump the middle jump must first be performed and then the second
